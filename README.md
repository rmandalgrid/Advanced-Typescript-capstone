# Advanced TypeScript Capstone Project

A project management application built with TypeScript demonstrating advanced concepts including classes, interfaces, decorators, generics, and drag-and-drop functionality.

## Features

- Create and manage projects with title, description, and team size
- Drag and drop projects between "Active" and "Finished" lists
- Form validation with custom utilities
- State management using Singleton pattern
- Component-based architecture

## Technologies

- TypeScript (v5.0.0)
- HTML5 (Drag & Drop API)
- CSS3
- ES6 Modules

## Installation

```bash
npm install
```

## Usage

**Development mode:**
```bash
npm run dev
```

**Build:**
```bash
npm run build
```

**Start production server:**
```bash
npm start
```

## Project Structure

```
src/
├── app.ts                 # Entry point
├── components/            # UI components
├── interfaces/           # TypeScript interfaces
├── models/               # Data models
├── state/                # State management
└── util/                 # Utilities (decorators, validation)
```
